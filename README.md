# astrbot_plugin_yysls

AstrBot 的燕云十六声排行榜插件。提供最新活动周期的总榜与详细榜图片。

## 指令

- `/燕云`：查看插件帮助
- `/排行榜`：获取两个当前副本的普通、挑战四榜总图
- `/排行榜 五人`：五人普通、挑战双榜
- `/排行榜 五人 挑战`：五人挑战详细榜
- `/排行榜 十人`：十人普通、挑战双榜
- `/排行榜 十人 挑战`：十人挑战详细榜

显式添加 `普通` 或 `挑战` 时只生成对应单榜。

每个榜单显示前 20 名的名次、队伍和用时。第 21 名以后按以下规则连续分段：

- 每段至少 5 队。
- 普通榜达到 3 秒跨度，或加入当前秒后达到 20 队时封段。
- 挑战榜达到 6 秒跨度，或加入当前秒后达到 20 队时封段。
- 最后一段不足 5 队时并入上一段。

聚合行显示名次和用时范围，例如 `21-26  7:01-7:07`。队伍名称最多保留约 12
个汉字的显示空间。

## 数据更新

插件不做后台轮询。每次执行排行榜命令时都会直接请求数据源的活动周期和四个实时榜接口，
并添加防缓存请求头与时间戳参数；插件本身不再缓存快照。图片顶部的“数据更新时间”来自接口
返回的 `snapshotTime`。数据源目前观察到约每小时生成一次新快照，因此网站尚未发布新快照时，
Bot 也无法提前取得新数据。

数据来自 [燕云十六声排行榜](https://yysls.rubysiu.cn/yysls/rank)，接口异常时插件会返回简短错误信息。

## 安装

将仓库放入 AstrBot 的 `data/plugins/astrbot_plugin_yysls`，安装依赖并重启 AstrBot：

```bash
pip install -r data/plugins/astrbot_plugin_yysls/requirements.txt
```

要求 AstrBot 4.16.0 或更高版本。

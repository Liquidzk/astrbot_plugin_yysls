# astrbot_plugin_yysls

AstrBot 的燕云十六声排行榜插件。提供最新活动周期的总榜与详细榜图片。

## 指令

- `/燕云`：查看插件帮助
- `/排行榜`：获取两个当前副本的普通、挑战四榜总图
- `/排行榜 五人`：五人普通、挑战双榜
- `/排行榜 五人 挑战`：五人挑战详细榜
- `/排行榜 十人`：十人普通、挑战双榜
- `/排行榜 十人 挑战`：十人挑战详细榜
- `/排行榜 五人 普通 7:11`：查询该用时在指定榜单中的名次

显式添加 `普通` 或 `挑战` 时只生成对应单榜。
五人、十人的普通和挑战四榜均支持 `分:秒` 时间查询；同一秒有多队时返回名次区间，
没有完全相同秒数时返回预计插入名次，并 @ 查询者。

每个榜单显示前 20 名的名次、队伍和用时。第 21 名以后按以下规则连续分段：

- 每段至少 5 队。
- 普通榜达到 3 秒跨度，或加入当前秒后达到 20 队时封段。
- 挑战榜达到 6 秒跨度，或加入当前秒后达到 20 队时封段。
- 最后一段不足 5 队时并入上一段。

聚合行显示名次和用时范围，例如 `21-26  7:01-7:07`。队伍名称最多保留约 12
个汉字的显示空间。

## 数据更新

`rank-backend` 中的更新工具每 5 分钟从燕云公开排行榜接口完整抓取当前第 2 期四榜，
并原子更新 `data/current-ranks.json`。插件优先读取 15 分钟内的本地快照，因此群命令
不会等待完整抓榜；快照缺失或过期时回退到
[燕云十六声排行榜](https://yysls.rubysiu.cn/yysls/rank) 的网站接口。两个来源都暂时
不可用时，插件会继续使用最后一份完整快照。

当前映射为五挑 60、五普 63、十挑 59、十普 62。换期时直接编辑
[`rank-backend/ranks.json`](rank-backend/ranks.json)，详见
[`rank-backend/README.md`](rank-backend/README.md)。

## 安装

将仓库放入 AstrBot 的 `data/plugins/astrbot_plugin_yysls`，安装依赖并重启 AstrBot：

```bash
pip install -r data/plugins/astrbot_plugin_yysls/requirements.txt
```

要求 AstrBot 4.16.0 或更高版本。
